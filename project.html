Project Information

  • Category: Mobile Application
  • Tech Stack: Android Studio, Kotlin, Retrofit, MotionLayout, ViewModel & LiveData, Coroutines & Flow, Navigation Component

Modern Weather App

This sleek and modern weather forecasting mobile app is developed in Kotlin using Android Studio. It delivers real-time weather updates with animated UI transitions via MotionLayout.

The app fetches current, hourly, and weekly forecasts using Retrofit and a weather API. ViewModel, LiveData, Coroutines, and Flow ensure asynchronous data loading and responsive behavior.

Highlights include location-based updates, UV index, sunrise/sunset times, humidity, visibility, and wind speed metrics, all packed into an intuitive interface navigated using the Navigation Component.
